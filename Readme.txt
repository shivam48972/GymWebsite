Steps to be followed for executing project-

1. make sure you have source code of project. if not then download from - 
https://github.com/shivam48972/GymWebsite

2. extract the zip file -
if you are using xampp then extract in htdocs folder
in case of wampp extract by creating www folder.

3. from source code select database file "gym.sql" and import into your database using phpmyadmin.

4. All changes done. You can execute website using localhost.

5. Congrats !
